# Dashboard de Viabilidade de Expansão Regional

Projeto de Business Intelligence desenvolvido para análise da viabilidade de expansão regional utilizando indicadores operacionais e critérios automatizados de decisão.

## Indicadores utilizados

- Volume médio de pedidos
- OTIF (On Time In Full)
- Custo por entrega

## Critérios de decisão

- Volume >= 4200
- OTIF >= 90%
- Custo < 18.5

## Ferramentas utilizadas

- Google Sheets
- Power BI (em desenvolvimento)

## Objetivo

Automatizar a análise GO/NO-GO para apoiar decisões de expansão regional.
